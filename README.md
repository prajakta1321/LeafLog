# Fast API :
 How to run :

 ```
cd leaflog-backend              # cd (folder_name)
```

start Virtual Environment :

```
.\.venv\Scripts\activate
```

Install FastAPI Uvicorn :

```
pip install fastapi uvicorn mysql-connector-python
```
Run the app :

```
uvicorn main:app --reload
```
